# API Documentation

## Overview

The AI Phone Automation Platform exposes APIs for managing AI agents, phone workflows, appointment scheduling, customer records, and analytics.

This documentation will be expanded as the backend API is implemented.

## Planned Endpoints

### Authentication

- POST /login
- POST /logout

### AI Agents

- GET /agents
- POST /agents
- PATCH /agents/{id}

### Calls

- GET /calls
- GET /calls/{id}

### Appointments

- GET /appointments
- POST /appointments

### Businesses

- GET /businesses
- PATCH /businesses/{id}
